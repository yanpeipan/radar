  <action>
    Create src/feeds.py implementing:

    1. generate_feed_id() - generates UUID for feed

    2. fetch_feed_content(url, etag=None, last_modified=None):
       - Uses httpx.get() with timeout=30.0
       - Returns (content, etag, last_modified, status_code)
       - Handles httpx.RequestError, httpx.TimeoutException
       - Status 304 = not modified, return None content

    3. parse_feed(content, url):
       - Uses feedparser.parse() on content
       - Returns (entries, bozo_flag, bozo_exception)
       - Bozo detection: feed.bozo is True for malformed XML
       - Extract entries from feed.entries
       - For each entry extract: title, link, guid (fallback to link or hash), pubDate, description, content

    4. generate_article_id(entry):
       - If entry.get('id'): return it
       - If entry.get('link'): return it
       - Fallback: SHA256(f"{link}:{pub_date}") hexdigest

    5. add_feed(url):
       - Fetch and parse feed first (validates URL is valid feed)
       - Create Feed object with id=uuid, name from feed.title, url
       - Insert into database
       - Return Feed object
       - Raises ValueError if already exists or parse fails

    6. list_feeds():
       - Query all feeds from database
       - Return list of Feed objects
       - Include article count per feed (SELECT COUNT from articles)

    7. get_feed(feed_id):
       - Get single feed by ID, return Feed or None

    8. remove_feed(feed_id):
       - Delete feed from database (ON DELETE CASCADE removes articles)
       - Return True if deleted, False if not found

    9. refresh_feed(feed_id):
       - Get feed from database
       - Fetch new content using stored etag/last_modified
       - Parse feed content
       - Insert new articles (IGNORE clause handles duplicates due to UNIQUE constraint)
       - Update feed's last_fetched, etag, last_modified
       - Return dict with new_articles count, updated_etag, updated_last_modified

    10. FeedNotFoundError exception class

    Import: uuid, hashlib, httpx, feedparser, from src.db import get_connection
    From src.models import Feed, Article
