<introduction>
## Gap Analysis

**Root Cause:** `add_feed()` in `src/feeds.py` blindly calls `fetch_feed_content()` + `parse_feed()` for all URLs. GitHub blob pages are HTML (JavaScript-rendered), not RSS feeds, so parsing fails with 403/422.

**Key Findings from Research:**
1. `is_github_blob_url()` exists in `src/crawl.py` - detects GitHub blob URLs
2. `detect_changelog_file()` returns None for explicit blob URLs (e.g., README.md) - need to fetch the specified path directly
3. `store_changelog_as_article()` requires `repo_id` FK to `github_repos` - must create/get github_repo entry first
4. `refresh_feed()` does not detect GitHub URLs - needs to call `refresh_changelog()` or equivalent for GitHub feeds

**Design Decisions (from D-01 through D-09):**
- D-01: Modify `add_feed()` to detect URL type at start
- D-02: GitHub blob URL -> changelog branch; RSS/Atom -> original RSS branch
- D-03: Use `is_github_blob_url()` for detection
- D-04: GitHub URL feed uses random UUID (same as RSS feed)
- D-05: GitHub URL stored as `feed.url`
- D-06: changelog content via `fetch_changelog_content()` stored as articles
- D-07: No changelog detected -> error, do NOT add feed
- D-09: `fetch --all` for GitHub URL feed calls refresh logic

**Critical Design Gap:** `store_changelog_as_article()` uses `INSERT OR IGNORE` with new article_id each call - subsequent refreshes create duplicate articles. Must fix to UPDATE existing article instead.
</introduction>

    def get_or_create_github_repo(owner: str, repo: str) -> GitHubRepo:
        """Get existing GitHub repo or create new entry if not exists.

        Args:
            owner: GitHub owner (user or org)
            repo: Repository name

        Returns:
            GitHubRepo object (existing or newly created)
        """
        conn = get_connection()
        try:
            cursor = conn.cursor()
            cursor.execute(
                "SELECT * FROM github_repos WHERE owner = ? AND repo = ?",
                (owner, repo),
            )
            row = cursor.fetchone()
            if row:
                return GitHubRepo(
                    id=row["id"],
                    name=row["name"],
                    owner=row["owner"],
                    repo=row["repo"],
                    last_fetched=row["last_fetched"],
                    last_tag=row["last_tag"],
                    created_at=row["created_at"],
                )
        finally:
            conn.close()

        # Create new entry
        repo_id = generate_repo_id()
        name = f"{owner}/{repo}"
        now = datetime.now(timezone.utc).isoformat()

        conn = get_connection()
        try:
            cursor = conn.cursor()
            cursor.execute(
                """INSERT INTO github_repos (id, name, owner, repo, last_fetched, created_at)
                   VALUES (?, ?, ?, ?, ?, ?)""",
                (repo_id, name, owner, repo, now, now),
            )
            conn.commit()
        finally:
            conn.close()

        return GitHubRepo(
            id=repo_id,
            name=name,
            owner=owner,
            repo=repo,
            last_fetched=now,
            last_tag=None,
            created_at=now,
        )
    ```

    def add_github_blob_feed(url: str, github_blob: tuple[str, str, str, str]) -> Feed:
        """Add a feed from GitHub blob URL, storing changelog content as article.

        Args:
            url: The GitHub blob URL
            github_blob: Tuple of (owner, repo, branch, path) from is_github_blob_url()

        Returns:
            The created Feed object.

        Raises:
            ValueError: If content fetch fails or no github_repo could be created.
        """
        owner, repo, branch, path = github_blob
        filename = path.split('/')[-1] if '/' in path else path

        # D-04, D-06: Ensure github_repo entry exists for store_changelog_as_article()
        github_repo = get_or_create_github_repo(owner, repo)

        # D-06: Fetch content for the explicit path
        content = fetch_changelog_content(owner, repo, filename, branch)
        if not content:
            # D-07: No content -> clear error, do NOT add feed
            raise ValueError(f"Failed to fetch content from {url}")

        # Store changelog as article with repo_id association
        source_url = f"https://raw.githubusercontent.com/{owner}/{repo}/{branch}/{filename}"
        store_changelog_as_article(
            repo_id=github_repo.id,
            repo_name=github_repo.name,
            content=content,
            filename=filename,
            source_url=source_url,
        )

        # D-04, D-05: Create feed entry with GitHub URL as url
        feed_id = generate_feed_id()
        now = datetime.now(timezone.utc).isoformat()
        feed_title = f"{owner}/{repo} / {filename}"

        conn = get_connection()
        try:
            cursor = conn.cursor()
            # Check if feed already exists
            cursor.execute("SELECT id FROM feeds WHERE url = ?", (url,))
            existing = cursor.fetchone()
            if existing:
                raise ValueError(f"Feed already exists: {url}")

            cursor.execute(
                """INSERT INTO feeds (id, name, url, etag, last_modified, last_fetched, created_at)
                   VALUES (?, ?, ?, ?, ?, ?, ?)""",
                (feed_id, feed_title, url, None, None, now, now),
            )
            conn.commit()
        finally:
            conn.close()

        return Feed(
            id=feed_id,
            name=feed_title,
            url=url,
            etag=None,
            last_modified=None,
            last_fetched=now,
            created_at=now,
        )
    ```

    **4. Add refresh_github_blob_feed() helper function:**
    ```python
    def refresh_github_blob_feed(feed: Feed, github_blob: tuple[str, str, str, str]) -> dict:
        """Refresh a GitHub blob URL feed by re-fetching the specific file.

        Args:
            feed: The Feed object to refresh
            github_blob: Tuple of (owner, repo, branch, path)

        Returns:
            Dict with refresh status.
        """
        owner, repo, branch, path = github_blob
        filename = path.split('/')[-1] if '/' in path else path
        repo_name = f"{owner}/{repo}"
        guid = f"changelog:{repo_name}:{filename}"

        # Find existing article by guid
        conn = get_connection()
        try:
            cursor = conn.cursor()
            cursor.execute("SELECT id FROM articles WHERE guid = ?", (guid,))
            row = cursor.fetchone()
            if not row:
                return {"changelog_found": False, "error": "Article not found for guid: " + guid}
        finally:
            conn.close()

        # Re-fetch content
        content = fetch_changelog_content(owner, repo, filename, branch)
        if not content:
            return {"changelog_found": True, "filename": filename, "error": "Failed to fetch content"}

        source_url = f"https://raw.githubusercontent.com/{owner}/{repo}/{branch}/{filename}"

        # Update existing article (uses same article_id)
        store_changelog_as_article(
            repo_id=None,  # Will update existing article's repo_id
            repo_name=repo_name,
            content=content,
            filename=filename,
            source_url=source_url,
        )

        now = datetime.now(timezone.utc).isoformat()
        conn = get_connection()
        try:
            cursor = conn.cursor()
            cursor.execute(
                "UPDATE articles SET content = ?, link = ?, pub_date = ? WHERE guid = ?",
                (content, source_url, now, guid),
            )
            conn.commit()
        finally:
            conn.close()

        return {"changelog_found": True, "filename": filename, "article_id": row["id"]}
    ```
