## Key Decisions

- D-01: 修改 add_feed() 内部处理 GitHub URL

- D-03: 复用 is_github_blob_url() 做 URL 检测
- D-04: GitHub URL feed 使用随机 UUID 作为 feed_id
- D-05: GitHub URL 本身作为 feed.url 存储
- D-06: changelog 内容存入 articles 表
- D-07: 未检测到 changelog → 返回明确错误，不添加 feed
- D-08: 网络错误透传原始信息
- D-09: fetch --all 时对 GitHub URL feed 调用 refresh_changelog()
