- priority() -> int: Return 50

- tag_parsers() -> List[TagParser]: Return [] (tag parsers loaded separately in Plan 02)
- parse_tags(article: Article) -> List[str]: Return [] (chaining implemented in Plan 02 via chain_tag_parsers)


Error handling for crawl: wrap in try/except, log.error on failure, return [].

- priority() -> int: Return 100 (highest, tried first)
- crawl(url: str) -> List[Raw]: Call parse_github_url(url) to get (owner, repo), then call fetch_latest_release(owner, repo). Return [release_data] as single-item list, or [] if no release. On exception, log error and return [].
- parse(raw: Raw) -> Article: Convert release dict to Article dict with fields: title (tag_name or name), link (html_url), guid (tag_name), pub_date (published_at), description (body), content (None).
- tag_parsers() -> List[TagParser]: Return [] (tag parsers loaded separately in Plan 02)
- parse_tags(article: Article) -> List[str]: Return [] (chaining implemented in Plan 02 via chain_tag_parsers)
