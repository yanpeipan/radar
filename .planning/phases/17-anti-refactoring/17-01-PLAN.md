  <files>src/db.py</files>
  <action>
    **Per D-01 (DB via context manager)**

    1. Add `from contextlib import contextmanager` at top of db.py
    2. Rename existing `get_connection` to `_get_connection` (internal function)
    3. Add new `get_db()` context manager after `_get_connection`:

    ```python
    @contextmanager
    def get_db():
        """Context manager for database connections.

        Yields a configured connection and ensures it is closed on exit.
        Usage:
            with get_db() as conn:
                cursor = conn.cursor()
                cursor.execute(...)
                conn.commit()
        """
        conn = _get_connection()
        try:
            yield conn
        finally:
            conn.close()
    ```

    3. Export `get_db` in addition to existing exports
  </action>
