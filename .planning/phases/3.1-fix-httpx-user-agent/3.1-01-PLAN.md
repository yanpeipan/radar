<objective>
Fix 403 Forbidden error when crawling qbitai.com by adding browser User-Agent header to httpx requests. This is a Phase 3 regression: httpx.get() uses default User-Agent 'python-httpx/<version>' which sites identify as bot and block.
</objective>

<intro>
Root cause identified in debug session: src/crawl.py line 125 uses httpx.get() without custom headers. Default User-Agent 'python-httpx/<version>' is blocked by qbitai.com. Fix: add browser-like User-Agent header to all httpx requests.
</intro>

@src/crawl.py
</context>
