    Create `src/cli/discover.py` implementing the `discover` CLI command:

    1. **Imports** (use absolute imports from src):
       - `import sys`, `import logging`
       - `import click`
       - `import uvloop`
       - `from rich.console import Console`
       - `from rich.table import Table`
       - `from src.discovery import discover_feeds, DiscoveredFeed`
       - `from src.cli import cli`

    2. **CLI command structure**:
       - Decorator: `@cli.command("discover")`
       - Argument: `@click.argument("url")`
       - Option: `@click.option("--discover-deep", default=1, type=click.IntRange(1, 10), help="Crawl depth for feed discovery (default: 1)")`
       - Context: `@click.pass_context`

    3. **Command function signature**:
       ```python
       def discover(ctx: click.Context, url: str, discover_depth: int) -> None:
       ```

    4. **Implementation** (inside command function):
       - If `discover_depth > 1`: Show "Deep crawling (depth > 1) is not yet implemented. Use depth=1 for single-page discovery." and return (Phase 37 handles deep crawling)
       - Call `uvloop.run(_discover_async(url))` to get list of DiscoveredFeed
       - Call `_display_feeds(feeds)` to display results
       - On exception: `click.secho(f"Error: {e}", err=True, fg="red")` and `sys.exit(1)`

    5. **Async implementation** (`_discover_async` function):
       - Accept `url: str` parameter
       - Return `list[DiscoveredFeed]`
       - Simply call and return `discover_feeds(url)`

    6. **Display function** (`_display_feeds` function):
       - Accept `feeds: list[DiscoveredFeed]`
       - If empty: `click.secho("No feeds discovered.", fg="yellow")` and return
       - Create Rich Table with columns: Type (width=8, style="dim"), Title (max_width=40), URL
       - For each feed:
         - Color code feed_type: rss="red", atom="green", rdf="blue"
         - Add row: colored feed_type, title or "—", url
       - Print table with `console.print(table)`
       - Print `click.secho(f"Discovered {len(feeds)} feed(s)", fg="green")`

    7. **Docstring** for command:
       ```
       Discover RSS/Atom/RDF feeds from a website URL without subscribing.

       Examples:

         rss-reader discover example.com
         rss-reader discover example.com --discover-deep 1
       ```

    IMPORTANT: Use `sys.exit(1)` on error, not `return 1`.
