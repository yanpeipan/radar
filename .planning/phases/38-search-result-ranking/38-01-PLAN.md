## Ranking Algorithm (from CONTEXT D-01)
```
final_score = 0.5 * norm_similarity + 0.3 * norm_freshness + 0.2 * source_weight

norm_similarity = (cos_sim - min_cos_sim) / (max_cos_sim - min_cos_sim)
norm_freshness = 1 - (days_ago / 30)
cos_sim = max(0.0, 1.0 - distance^2 / 2)  # Already computed in search.py
source_weights = {'openai.com': 1.0, 'arxiv.org': 0.9, 'medium.com': 0.5, 'default': 0.3}
```

## Source Weight Domain Matching (D-02)
- Domain extracted from article URL via `urllib.parse.urlparse`
- Suffix match: `blog.openai.com` matches `openai.com`
- Falls back to `default` if no match

## Pre-v1.8 Exclusion (D-03)
- `search_articles_semantic()` returns results with `sqlite_id=None` for articles without embeddings
- These are excluded from ranked results (cannot calculate freshness without pub_date)

## Integration Point (D-04)
- Ranking called AFTER `search_articles_semantic()` and BEFORE `format_semantic_results()`
- Flow: `search_articles_semantic() -> rank_semantic_results() -> format_semantic_results()`

## CLI Output (D-05)

    The function signature:
    ```python
    def rank_semantic_results(results: list[dict[str, Any]], top_k: int = 10) -> list[dict[str, Any]]:
    ```

    Algorithm steps:

    1. **Filter pre-v1.8 articles**: Skip any result where `sqlite_id` is None (these articles have no ChromaDB embedding and cannot be ranked)

    2. **Look up pub_date from SQLite**: For each result with `sqlite_id`, call `get_article(sqlite_id)` to get the article's `pub_date`

    3. **Calculate cos_sim for each result**: If `distance` is not None: `cos_sim = max(0.0, 1.0 - (distance * distance / 2.0))`. If distance is None, use 0.0

    4. **Min-max normalize similarity**: Find min and max cos_sim across all results. If max > min: `norm_sim = (cos_sim - min) / (max - min)`. Otherwise, use 1.0

    5. **Calculate freshness**: `days_ago = (datetime.now(timezone.utc) - pub_date.replace(tzinfo=timezone.utc)).days`, then `freshness = max(0.0, 1 - days_ago / 30)`. If article has no pub_date, use 0.0

    6. **Get source_weight**: Extract domain from URL via `urllib.parse.urlparse`. Match against `{'openai.com': 1.0, 'arxiv.org': 0.9, 'medium.com': 0.5, 'default': 0.3}` using suffix match (e.g., `blog.openai.com` matches `openai.com`)

    7. **Calculate final_score**: `final_score = 0.5 * norm_sim + 0.3 * freshness + 0.2 * source_weight`

    8. **Sort and slice**: Sort results by `final_score` descending, return top `top_k` results

    9. **Preserve all original keys**: Each result dict in the output must include all original keys PLUS `norm_similarity`, `norm_freshness`, `final_score`, and `cos_sim`
