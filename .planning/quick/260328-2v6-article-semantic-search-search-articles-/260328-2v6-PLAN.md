    ```python
    def format_articles(items: list, mode: str = "list", verbose: bool = False) -> list[dict[str, Any]]:
        """Unified article list formatter for all display modes.

        Args:
            items: List of articles (mode-dependent type)
            mode: 'list' for list_articles, 'fts' for search_articles, 'semantic' for rank_semantic_results
            verbose: Include full details if True

        Returns:
            List of dicts with unified fields: id, title, source, date, score
        """
        if mode == "list":
            return _format_list_items(items, verbose)
        elif mode == "fts":
            return _format_fts_items(items, verbose)
        elif mode == "semantic":
            return _format_semantic_items(items, verbose)
        else:
            raise ValueError(f"Unknown mode: {mode}")
    ```

    实现三个内部格式化函数：

    _format_list_items(items, verbose): 处理list_articles的ArticleListItem列表
    - id: article.id[:8] (non-verbose) or article.id (verbose)
    - title: article.title[:60]
    - source: article.feed_name[:15]
    - date: article.pub_date[:10] or "-"
    - score: "LIST"

    _format_fts_items(items, verbose): 处理search_articles的ArticleListItem列表
    - id: ""
    - title: article.title[:60]
    - source: article.feed_name[:15]
    - date: article.pub_date[:10] or "-"
    - score: "FTS"

    _format_semantic_items(items, verbose): 处理rank_semantic_results的dict列表
    - id: sqlite_id[:8] or "" (non-verbose), full sqlite_id or "" (verbose)
    - title: title[:60]
    - source: domain from url[:15] or "-"
    - date: "-"
    - score: norm_similarity * 100 as percentage

    修改format_semantic_results()函数内部实现，委托给format_articles(mode="semantic")：
