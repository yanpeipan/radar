## What was done

### Task 1: Add score field to rank_semantic_results output
- Added `score` field = `final_score` (normalized 0-1 value) to rank_semantic_results output
- Updated `_format_semantic_items` to use `score` field when present, fall back to `norm_similarity` for backward compatibility

### Task 2: Create rank_fts_results and rank_list_results functions
- Added `rank_fts_results(articles: list[ArticleListItem]) -> list[dict[str, Any]]`
  - Returns all article fields plus `score=1.0` (fixed, no similarity metric for FTS)
- Added `rank_list_results(items: list[ArticleListItem]) -> list[dict[str, Any]]`
  - Returns all item fields plus `score=1.0` (fixed, no similarity metric for list)

### Task 3: Update format functions to use score field
- Updated `_format_list_items` to handle dict input (from rank_list_results) and use `score` field when present
- Updated `_format_fts_items` to handle dict input (from rank_fts_results) and use `score` field when present
- Both fall back to "LIST"/"FTS" strings for backward compatibility when `score` not present
