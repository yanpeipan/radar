## What Was Done

### Task 1: search_articles_fts and search_articles_semantic
Added `rerank: bool = False` parameter to both functions.

When `rerank=True`:
1. Skip initial `combine_scores` (rerank will override ce_score anyway)
2. Call `asyncio.run(rerank(query, articles, limit))` via ThreadPoolExecutor
3. After rerank sets ce_score on each article, recompute final_score using `combine_scores` with appropriate weights:
   - FTS: alpha=0.3, beta=0.3, gamma=0.0, delta=0.2
   - Semantic: alpha=0.3, beta=0.3, gamma=0.2, delta=0.0

### Task 2: CLI article_search
Removed:
- `import asyncio`
- `RERANK_FACTOR` logic
- `do_score = not rerank` logic
- `asyncio.to_thread(rerank, ...)` call block

CLI now simply passes `rerank=rerank` directly to search functions.
