## Task: Loop 50 iterations of v2 report, fix errors

**Action:** Write a script that:
1. Runs `uv run feedship --debug report --template v2 --since 2026-04-01 --until 2026-04-08` up to 50 times
2. Captures stderr and exit codes
3. Groups errors by type (LLM rate limit, import errors, template errors, etc.)
4. For each unique error, fixes the root cause in the code
5. Common fixes expected: LLM failures (graceful degradation), empty section handling, translation chain issues

**Verify:** Run the command 3 times after fixes, confirm no crashes.
