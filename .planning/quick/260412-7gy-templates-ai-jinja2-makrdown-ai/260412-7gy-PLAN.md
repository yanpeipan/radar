Key design decisions (per D-01 to D-04):
- Uses `{%-` whitespace strip to avoid excessive blank lines
- Uses `{%- else -%}` Jinja2 built-in to detect empty iterables
- Articles accessed via `topic.dimensions[dim]` nested iteration (NOT `topic.sources` — per RESEARCH finding)
- Each layer letter prefixed (A-E) matching example format
- Empty state exactly "（本期暂无相关来源）"
- `director_note` optional: only renders if provided
